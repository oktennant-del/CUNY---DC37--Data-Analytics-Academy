This project explores a public health dataset on diabetes outcomes, focusing on how clinical characteristics differ across patient groups. Using Python for data cleaning, descriptive statistics, and visualization, the analysis compares hospital stay lengths between patients who received diabetes medication and those who did not. The project demonstrates practical skills in data wrangling, exploratory analysis, and health‑focused interpretation, offering a clear, reproducible workflow for examining patterns in chronic disease management.

Source: 
https://raw.githubusercontent.com/niteen11/DataAnalyticsAcademy/master/Python/dataset_diabetes/diabetic_data.csv

How to Run this Project

Clone the repository

git clone 

cd

Install required Python packages

pip install pandas matplotlib seaborn

Open the Jupyter notebook

jupyter notebook

Then open the notebook file (diabetes_analysis.ipynb).

You can also run it in Google Colab by uplaoding the notebook or opening it directly from GitHib.

Run all cells

The notebook will automatically:

1. Load the diabetes dataset
2. Clean the data (replace "?", convert age ranges)
3. Create comparison groups
4. Generate descriptive statistics
5. Produce a visualization
6. Output a brief conclusion
   
Extend the analysis


You can modify the comparison grous, add new visualizations, or explore additional public health indicators.
